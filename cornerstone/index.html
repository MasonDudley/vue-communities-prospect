The VUE at Cornerstone

A cleaner, calmer student setup with 3BR / 3BA privacy built in.

Cornerstone is the focused option in the VUE Communities lineup. It offers one furnished 3-bedroom, 3-bathroom layout in a residential neighborhood off Greenview Drive, with each resident getting a private ensuite bathroom and walk-in closet.
